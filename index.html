League Rules
Format

Weekly 2-man best ball match play. Each matchup is played over 9 holes.

Scoring

Lowest net score from each team counts on every hole. Winning more holes wins the match.

Handicaps

Score entry uses the Twin Hills course data and calculates 9-hole course handicaps for the selected side.

Subs

Use the actual player names for the week so player stats stay accurate. Subs can be entered manually in the scorecard.

Playoff Picture

The dashboard currently shows the top 4 as playoff position teams. This can be adjusted once the final playoff format is locked in.

Backups

Commissioners should export a backup after entering weekly results so standings and stats can be restored if needed.
